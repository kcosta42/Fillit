>
> That's why some parts may be strange

About
-----

Second project of School 42 Paris

> A recurring problem in programming: the search for solution, optimal among a large number of possibilities, within a reasonable time. In the case of this project, it will be a question of arranging Tetriminos between them and to determine the smallest square possible.

Installation
------------
run `make fclean && make`

Usage
-----
`./fillit file`
* file: a valid file describing tetriminos

#### Example
```
./fillit samples/sample.fillit2
ADDD
ABDC
ABCC
ABBC
```
